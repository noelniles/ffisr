# Modern Research Resources for the FFISR White Paper

This is a curated list of sites/resources to ground claims and related work with recent literature.

## Core Paper Repositories

1. arXiv Computer Vision (cs.CV): https://arxiv.org/list/cs.CV/recent
2. arXiv Networking / Information Theory (cs.NI, cs.IT):
   - https://arxiv.org/list/cs.NI/recent
   - https://arxiv.org/list/cs.IT/recent
3. CVF Open Access (CVPR/ICCV/ECCV papers): https://openaccess.thecvf.com/

## Tracking / Detection Benchmarks

1. MOTChallenge benchmark: https://motchallenge.net/
2. VisDrone benchmark and downloads: http://aiskyeye.com/
3. Papers With Code (MOT + UAV tracking leaderboards): https://paperswithcode.com/

## High-Value Topics to Cite (2021+)

- Real-time transformer detectors (RT-DETR and variants)
- Detection-to-tracking pipelines (ByteTrack, BoT-SORT)
- Task-oriented / semantic communications for edge AI
- UAV perception under constrained communication
- Age-of-information or freshness metrics under lossy links

## Suggested Citation Hygiene

- Prefer peer-reviewed conference/journal versions where available.
- Keep arXiv IDs for fast traceability.
- Verify benchmark claims against official leaderboard pages.
- Separate communication-energy proxies from true platform power measurements.
